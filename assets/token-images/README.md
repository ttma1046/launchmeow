# Token Images

Place meme token images in this directory.

## Supported Formats
- JPG/JPEG
- PNG
- GIF
- WEBP

## Usage
The bot will randomly select one image from this folder when launching a new token.

## Image Sources
Make sure you have the right to use any images you place here:
- Create your own images
- Use royalty-free images from sites like Unsplash, Pexels
- Use AI-generated images
- Only use images you have permission to use

## Recommended
- Image size: 512x512 or 1024x1024
- Format: PNG or JPG
- Keep file sizes reasonable (<5MB)
